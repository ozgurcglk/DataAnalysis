# DataAnalysis
Data Analysis with Python 

Görselleştirilmiş veriler ilgili task altındaki "graphs" dosyalarının altındadır.
Çalıştırmak için requirements.txt dosyasının içerisindeki gereksinimler sistemde yüklü olmalıdır.

Repository clone edildikten sonra run_task1.py ve run_task2.py dosyaları run edilerek yeni grafikler de görülebilir.

Ek olarak - analizlerin sonuçları cmd'ye de yazdırılmaktadır.
